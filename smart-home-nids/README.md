# 🛡️ Smart Home NIDS — Network Intrusion Detection System

A **production-quality, machine-learning-based Network Intrusion Detection System** designed to detect malicious network traffic generated by IoT devices in smart home environments.

Built on the **CIC IoT-2023** dataset (Canadian Institute for Cybersecurity) with a **Random Forest classifier**, **Streamlit dashboard**, **SQLite logging**, **SHAP explainability**, and **multi-channel alerting**.

## ✨ Features

### Machine Learning
- **Random Forest classifier** with balanced class weights for imbalanced data
- **9 attack categories**: BENIGN, DDoS, DoS, Mirai, Recon, Spoofing, BruteForce, WebAttack, Malware
- **18 network flow features** selected via Random Forest importance + Mutual Information
- **RobustScaler** for outlier-heavy traffic data
- **SHAP explainability** (TreeExplainer) for model transparency
- **Prediction confidence** scores with severity classification

### Dashboard
- 🌙 **Dark glassmorphism theme** with Inter font and gradient accents
- 📊 **Real-time KPI cards** — total flows, threats, devices, alerts
- 🎯 **Threat detection gauge** — visual severity indicator
- 📈 **Interactive Plotly charts** — pie, bar, line, heatmap, gauge
- 🔴 **Live monitoring** — CSV upload, manual input, traffic simulation
- 📋 **Searchable logs** — filter by type, severity, IP, date range
- 📥 **CSV/JSON export** — download detection history
- 🖥️ **Device risk scoring** — per-device threat assessment
- ⚡ **Auto-refresh** — configurable interval

### Alerting
- 🖥️ Desktop notifications (plyer)
- 📧 Email alerts (SMTP)
- 📱 Telegram webhook
- ⏱️ Rate limiting (prevent alert storms)
- 🎚️ Severity-based routing

### Infrastructure
- 🐳 Docker & Docker Compose support
- 🔄 GitHub Actions CI/CD
- 🗄️ SQLite persistence (detections, devices, alerts, statistics)
- 🧪 Comprehensive test suite (pytest)
- 📝 PEP 8 compliant, type-hinted, docstrings throughout

## 🚀 Quick Start

### 1. Clone & Setup

```bash
git clone https://github.com/yourusername/smart-home-nids.git
cd smart-home-nids
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Prepare Data

Place CIC IoT-2023 CSV files in `data/raw/`, or the pipeline will auto-discover them from `../train/`, `../test/`, `../validation/`.

### 3. Run Preprocessing

```bash
python src/preprocessing.py
```

**Output**: `data/processed/processed_train.csv`, `data/processed/processed_test.csv`, model artifacts in `models/`.

### 4. Train Model

```bash
python src/train_model.py
```

**Output**: `models/rf_model.pkl`, `reports/training_report.json`, `plots/confusion_matrix.png`.

### 5. Launch Dashboard

```bash
streamlit run app.py
```

Open `http://localhost:8501` in your browser.
